# Front_end_web3

Base frontend prête à développer avec **React + TypeScript + Vite**.

## Installation

```bash
npm install
```

## Démarrage

```bash
npm run dev
```

## Structure

```txt
src/
├── assets/
├── components/
│   ├── App/
│   ├── Factures/
│   ├── Login/
│   ├── Menu/
│   └── UserList/
├── contexts/
├── lang/
├── index.css
└── main.tsx
```
